# AGENTS.md

## Goal
Build a mobile-first WebAR romantic 3D memory website with image-target scanning and a cinematic particle-based gallery scene.

## Rules
- Do not replace real 3D with CSS-only fake effects.
- Preserve the scanner -> loading -> 3D intro -> gallery -> top-view flow.
- Prioritize visual polish and smooth transitions.
- Optimize for mobile portrait performance.
- Keep code modular and readable.

## Commands
- Install dependencies
- Run dev server
- Run lint
- Run type-check
- Build project before finalizing

## Visual priorities
1. Scanner experience must feel premium
2. Particle sphere must look dense and beautiful
3. Camera transitions must feel cinematic
4. Gallery must feel like floating memories in space
5. Title text must remain readable and elegant

## Content
Use centralized config for recipient name, title, subtitle, loading text, target image path, and photo list.
